Unified API

Every US property. One endpoint. REST, GraphQL, and MCP interfaces. Address lookup, parcel search, zoning, permits — all normalized to a single schema.

MCP Native

Built for Model Context Protocol. Your agent calls ParcelData like any other tool. Property lookup, zoning checks, feasibility analysis — all as MCP tools.

# Lookup a property by address
curl https://api.parceldata.ai/v1/properties/address \
  -H "Authorization: Bearer pk_live_..." \
  -d '{"street":"123 Main St","city":"Austin","state":"TX"}'

# Response
{
  "address": "123 Main St, Austin, TX 78701",
  "zoning": "R1-SF",
  "lot_sqft": 8500,
  "bedrooms": 3,
  "tax_assessed": 425000,
  "data_quality": { "score": 0.92 }
}
from parceldata import ParcelData

pd = ParcelData(api_key="your-key")

# Lookup a property
prop = pd.lookup("123 Main St, Austin, TX")

print(prop.zoning)        # "R1-SF"
print(prop.lot_sqft)      # 8500
print(prop.data_quality)  # 0.92
print(prop.tax_assessed)  # 425000

# Search with filters
results = pd.search(
    city="Austin",
    zoning="R1-SF",
    min_sqft=2000
)
// MCP Server Configuration
{
  "server": "@parceldata/mcp",
  "tools": [
    "property_lookup",
    "property_search",
    "zoning_check",
    "permit_history",
    "get_comparables",
    "check_development_feasibility"
  ]
}

// Agent calls it naturally:
// "Look up 123 Main St in Austin, TX
//  and check development feasibility"
query PropertyLookup {
  property(address: "123 Main St, Austin, TX") {
    address
    zoning
    lot_sqft
    bedrooms
    year_built
    tax_assessed
    permits {
      type
      status
      date
    }
    data_quality {
      score
      freshness_hours
      sources
    }
  }
}

MCP Native

Your agent already knows
how to use it

ParcelData exposes every capability as an MCP tool. Any MCP-compatible AI agent can call property lookups, run zoning checks, and analyze markets — naturally.

property_lookup property_search zoning_check permit_history get_comparables estimate_value
mcp_config.json
{
  "mcpServers": {
    "parceldata": {
      "command": "npx",
      "args": ["@parceldata/mcp"],
      "env": {
        "PARCELDATA_API_KEY": "pk_live_..."
      }
    }
  }
}
